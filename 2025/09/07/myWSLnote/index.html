WSL/Linux 常用命令

初始設定

Root 密碼設定

基本命令格式

檔案與目錄操作

ls - 列出檔案和目錄

  • -a 列出隱藏檔案
  • -h 顯示檔案大小單位
  • ls -la 列表顯示隱藏檔案

快捷鍵：

  • <Ctrl+L> 清除畫面
  • <Ctrl+C> 終止

cd - 變更目錄

特殊目錄：

  • . 當前目錄
  • .. 上級目錄
  • ~ HOME目錄

mkdir - 建立目錄

  • -p 表示多層級目錄建立

touch - 建立檔案

檔案檢視

more 操作：

  • 空格鍵：下翻頁
  • b：上翻頁
  • q：退出

cp - 複製檔案

  • -r 用於複製資料夾（遞迴複製）

mv - 移動檔案

rm - 刪除檔案

  • -r 刪除資料夾
  • -f 強制刪除

萬用字元：

  • test* 匹配任意以test開頭的內容
  • *test* 匹配任何包含test的內容

檔案搜尋與查找

which - 尋找程式檔案位置

find - 搜尋檔案

grep - 關鍵字過濾

  • -n 在結果中顯示匹配行的行號
  • 關鍵字含空格或特殊符號時建議用雙引號包圍

wc - 統計檔案資訊

  • -c 統計byte數量
  • -m 統計字元數量
  • -l 統計行數
  • -w 統計單字數

進階操作

管道符 |

echo - 輸出內容

反引號

被反引號包圍的內容會被作為命令執行，而非普通字元

重定向符

tail - 檢視檔案尾部

  • -f 持續追蹤
  • -num 檢視尾部多少行，預設10行

範例：tail -5 new.txt

sudo - 提升權限

使用者與群組管理

以下命令需要root權限

群組管理

使用者管理

  • -g 指定使用者的群組，不指定則建立同名群組並自動加入
  • -d 指定使用者HOME路徑，預設在 /home/使用者名
  • -r 刪除使用者的HOME目錄

檔案權限管理

權限資訊解讀

  • 第一位：-表示檔案，d表示資料夾，l表示軟連結
  • 第二部分：所屬使用者權限
  • 第三部分：所屬群組權限
  • 第四部分：其他使用者權限

範例：drwxr-xr-x

權限意義：

  • r 檔案：可檢視檔案內容；資料夾：可檢視資料夾內容（如ls命令）
  • w 檔案：可修改檔案；資料夾：可在資料夾內建立、刪除、重新命名等操作
  • x 檔案：可將檔案作為程式執行；資料夾：可變更工作目錄到此資料夾（cd進入）

chmod - 修改權限

  • -R 對資料夾全部內容套用相同操作

符號表示法：

  • u表示user
  • g表示group
  • o表示other

數字表示法：

  • r記為4，w記為2，x記為1
  • 0表示沒有任何權限
  • 5表示有r和x權限（4+1）
  • 7有全部權限（4+2+1）

範例：751表示rwx(7)r-x(5)--x(1)

chown - 修改擁有者

  • 只適用於root使用者執行
  • -R 對資料夾全部內容套用相同規則
  • :用來分隔使用者和使用者群組

命令歷史與快捷鍵

命令歷史

快捷鍵

  • <Ctrl+D> 退出或登出
  • <Ctrl+R> 輸入內容匹配歷史命令
  • <Ctrl+A> 跳到開頭
  • <Ctrl+E> 跳到結尾
  • <Ctrl+左方向鍵> 向左跳一個單字
  • <Ctrl+右方向鍵> 向右跳一個單字
  • <Ctrl+L> 清空終端內容

套件管理

apt - 套件管理工具

需要root權限

  • apt install wget 安裝wget
  • apt remove wget 移除wget
  • apt search wget 搜尋wget

系統服務管理

systemctl - 服務控制

  • start 啟動
  • stop 關閉
  • status 檢視狀態
  • enable 開啟開機自啟
  • disable 關閉開機自啟

連結與快捷方式

ln - 建立連結

  • -s 建立軟連結
  • 參數1：被連結的檔案或資料夾
  • 參數2：要連結去的目的地

日期與時間

date - 日期顯示與格式化

格式化字串：

  • %Y 年
  • %y 年份後兩位數字
  • %m 月份 (01-12)
  • %d 日 (01-31)
  • %H 小時 (00-23)
  • %M 分鐘 (00-59)
  • %S 秒 (00-59)
  • %s 自1970-01-01 00:00:00 UTC到現在的秒數（時間戳）

範例：

日期計算單位：

  • year年、month月、day天、hour小時、minute分鐘、second秒

時間同步

網路相關

IP位址檢視

特殊IP：

  • 127.0.0.1 指代本機
  • 0.0.0.0 用於指代本機或在埠口中使用綁定來確定關係，或表示所有IP

修改主機名

ping - 網路連通性測試

  • -c 檢查的次數，不使用-c表示無限次持續檢查

wget - 檔案下載

  • -b 可選，後台下載，會將日誌寫入到當前工作目錄的wget-log檔案

curl - HTTP請求工具

  • -O 用於下載檔案，當url是下載連結時，可以使用此選項儲存檔案

nmap - 埠口掃描

netstat - 網路狀態檢視

埠口範圍說明

  • 公認埠口 1~1023：用於系統內建或常用知名軟體綁定使用
  • 註冊埠口 1024~49151：用於鬆散綁定使用（使用者自定義）
  • 動態埠口 49152~65535：用於臨時使用（多用於出口）

程序管理

ps - 程序檢視

  • -e 顯示全部的程序
  • -f 以完全格式化的形式展示資訊（展示全部資訊）

kill - 關閉程序

  • -9 表示強制關閉程序

top - 系統監控

透過top命令檢視CPU、記憶體使用情況，預設每5秒重新整理一次

選項說明：

  • -p 只顯示某個程序的資訊
  • -d 設定重新整理時間，預設5秒
  • -c 顯示產生程序的完整命令，預設是程序名
  • -n 指定重新整理次數，比如top -n 3，重新整理輸出3次後退出
  • -b 以非互動非全螢幕模式運行，以批次方式執行top
  • -i 不顯示閒置(idle)或殭屍(zombie)程序
  • -u 查找特定使用者啟動的程序

互動式操作（非-b選項啟動）：

  • h鍵：顯示幫助畫面
  • c鍵：顯示產生程序的完整命令，等同於-c參數
  • f鍵：選擇需要展示的項目
  • M鍵：根據駐留記憶體大小(RES)排序
  • P鍵：根據CPU使用百分比大小進行排序
  • T鍵：根據時間/累計時間進行排序
  • E鍵：切換頂部記憶體顯示單位
  • e鍵：切換程序記憶體顯示單位
  • l鍵：切換顯示平均負載和啟動時間資訊
  • i鍵：不顯示閒置或無用的程序，等同於-i參數
  • t鍵：切換顯示CPU狀態資訊
  • m鍵：切換顯示記憶體資訊

系統監控

df - 硬碟使用情況

  • -h 以更加人性化的單位顯示

iostat - I/O統計（需安裝sysstat）

  • -x 顯示更多資訊
  • num1 重新整理間隔
  • num2 重新整理幾次

sar - 系統活動報告

  • -n 檢視網路，DEV表示檢視網路介面
  • num1 重新整理間隔（不填就檢視一次結束）
  • num2 檢視次數（不填無限次數）

環境變數

env - 檢視環境變數

使用環境變數

export - 設定環境變數

永久設定：

  • 針對當前使用者生效：配置在 ~/.bashrc 檔案中
  • 針對所有使用者生效：配置在 /etc/profile 檔案中

檔案傳輸工具

lrzsz - 檔案傳輸工具

檔案壓縮與解壓

tar - 打包壓縮

選項說明：

  • -c 建立壓縮檔案
  • -v 顯示壓縮、解壓過程，用於檢視進度
  • -x 解壓模式
  • -f 要建立的檔案，或要解壓的檔案，-f選項必須在所有選項中位置處於最後一個
  • -z gzip模式，不使用-z就是普通的tarball模式
  • -C 選擇解壓目的地，用於解壓模式

範例：

zip/unzip - ZIP壓縮

  • -r 包含資料夾時需要帶
  • -d 表示指定解壓去的位置
