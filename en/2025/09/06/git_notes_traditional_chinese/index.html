IDEA快捷操作

主要入口

  • VCS選單 → Git相關操作
  • 右下角分支 → 切換/建立分支
  • Ctrl+K → 快速提交
  • Ctrl+Shift+K → 推送

常用操作

  • Add → 新增到暫存區
  • Commit → 提交到本地
  • Push → 推送到遠端
  • Pull → 從遠端拉取
  • Merge → 合併分支

常用Git指令速查

指令 功能
git status 檢視檔案狀態
git add . 新增所有變更
git commit -m "msg" 提交變更
git push 推送到遠端
git pull 從遠端拉取
git branch 檢視分支
git checkout -b 分支名 建立並切換分支
git merge 分支名 合併分支
git log --oneline 檢視提交歷史
git reset --hard HEAD^ 回退一個版本

檔案狀態標識

命令列

  • Untracked → 未追蹤檔案
  • Modified → 已修改檔案
  • Staged → 暫存區檔案

IDEA

  • 🟢 綠色 → 已新增到Git
  • 🔴 紅色 → 未新增到Git/衝突
  • ⚪ 灰色 → 已忽略檔案
  • 🟡 黃色 → 已修改檔案
